# 简介
1.单文件 html 搭配 DreamScene2 壁纸播放软件可实现单文件交互  
2.默认内置 base64 图片转码  
3.默认开启桌面交互功能,可在关键词 Heart-Yang 处 herf 标签(超链接)添加自定义跳转链接
